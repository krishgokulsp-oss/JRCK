# JRCK – Jawa Riders Club Kerala
## Website Package

### Files Structure
```
jrck-website/
├── index.html          ← Home Page
├── about.html          ← About JRCK
├── events.html         ← Events & Rides
├── team.html           ← Core Team
├── faq.html            ← FAQ (Accordion)
├── blog.html           ← Blog Grid
├── contact.html        ← Contact & Join Form
├── assets/
│   ├── css/
│   │   └── style.css   ← Main Stylesheet
│   └── js/
│       └── script.js   ← All Interactions
└── README.md
```

### Deployment
1. Upload all files to your hosting provider (maintain folder structure)
2. Point your domain to the root folder containing index.html
3. Add your JRCK logo PNG as /assets/images/logo.png for full logo display

### Customisation
- Replace placeholder phone/email with real details in all pages
- Update social media links in footer (Instagram, Facebook, YouTube, WhatsApp)
- Add real team member photos to /assets/images/team/
- Update Google Maps embed key/location as needed

### Colour Scheme
- Primary Orange: #F67E2A (RGB 246, 126, 42)
- Background Black: #000000
- Fonts: Bebas Neue (display), Barlow Condensed (headings), Barlow (body)
